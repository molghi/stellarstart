# Modern Website: StellarStart

A responsive, multi-page website built with Vue, with a clean and modern layout.

## Technologies Used:

- **Vue:** component-based architecture, props, events, ref, directives, lifecycle hooks, Vue Router, Composition API, and clean data separation with logic in the script and markup structure in the template;

- **Plain CSS:** flexbox, grid, hover animations, and responsive design with media queries.

## Features:

- Fully responsive design, hamburger menu for small screens, Swiper slider (_Home page_), filtering tabs (_Portfolio page_).
